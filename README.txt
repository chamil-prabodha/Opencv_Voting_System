--------------------Vote Counting Demo-----------------------------
Instructions:
1.  Run Main.py
2.  Select the ballot paper folder and the party sign folder attached with the project
3.  Click count votes button and wait for the program to finish (don't mind the progressbar)
4.  Console output will show the end of the program
5.  Click write to file to get the output to a file

Prerequisits:
1.  Opencv 3.0.0
2.  Pillow
3.  Tkinter
